Описание внутреннего содержания:

Папка "BOM" - Шаблоны ПЭ3 в формате xlt.

Папка "Fonts" - Шрифты, используемые при разработке Э3, ПЭ3, ПП.

Папка "Project" - Шаблоны пустых проектов для ускоренного старта разработки проекта.

Папка "SCH_1" - Форматки листов Э3 без подписи ПЗ.

Папка "SCH_5" - Форматки листов Э3 с подписью ПЗ.

Папка "Settings" - Настройки Altium Designer для быстрого разворачивания или восстановления рабочего места.

Папка "View" - Настройки видов ПП для печатного редактора в 2D и 3D видах.
